## Overview
This website is a simple personal portfolio created to document my journey while learning web development.  
The website introduces who I am, what I am currently learning, and some small projects I created while studying HTML and CSS.

The goal of this website is to practice building clean layouts, organizing website structure using semantic HTML, and styling pages using CSS.

This project also serves as a starting point for my future web development portfolio.
